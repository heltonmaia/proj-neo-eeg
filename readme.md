# Neo-EEG project
Application for interpreting and displaying EEG signals 

## 📁 Project Structure

```
proj-neo-eeg/
├── processing/             # signal processing & display
│   ├── openbci_gui/        # OpenBCI GUI configuration
│   └── libraries/          # dependencies for the processing application
│
└── microcontrollers/       # firmware for EEG hardware
    ├── libraries/          # dependencies for sensor interfacing
    └── code/               # firmware source code
```
